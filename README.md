# 🎮 Water Ring Toss

Recreación del clásico juguete de aros acuáticos de los 90s.

## 🕹️ Jugar
👉 **[JUGAR AQUÍ](https://TU-USUARIO.github.io/water-ring-toss/)**

## 🎯 Controles

| Acción | Teclado | Móvil |
|--------|---------|-------|
| Chorros de agua | A S D F G | Botones de colores |
| Inclinar horizontal | ← → | Botones morados / Giroscopio |
| Inclinar vertical | ↑ ↓ | Botones amarillos / Giroscopio |
| Reiniciar | R | Botón ↺ |

## ✨ Características
- 🌊 Física de agua realista
- 🎯 20 aros en 4 colores
- 🔥 Sistema de combos por color
- 📱 Compatible con móvil (giroscopio)
- 📊 Pantalla de resumen final
